
# Homelab 

Pequeño homelab casero usando obviamente mi favorito "Debian". Este proyecto integra una VPN Mesh mediante Tailscale para acceso remoto seguro y tambien utiliza Docker para desplegar servicios, en este caso Open WebUI y Ollama. Esto permite la administración remota por protocolo SSH, la ejecución de modelos de IA locales con una interfaz bonita y amigable ademas el acceso seguro a recursos de la red doméstica
